競馬予想アプリ 公開URL版 v2

この版でできること
- 前日版 / 当日版の整理
- ChatGPT用テキストの自動生成
- 各レースの馬一覧登録
- 推奨馬の自動表示
- 信頼度集計
- 条件集計
- 保存一覧

推奨馬ロジック
- 判定キー：競馬場 × 距離 × 芝ダ × 分類 × 信頼度
- 表示条件：複勝率90%以上、母数10以上

大事な点
- 静的HTML版なので Netlify / GitHub Pages / Cloudflare Pages に置きやすいです
- 保存はブラウザ内です
- PCとiPhoneで自動共有はされません
